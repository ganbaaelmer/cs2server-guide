Хэрхэн холбогдох вэ?

Ердөө 3 алхам. Эхний удаад л тохируулна — дараа нь шууд ордог.

1

ZeroTier програм суулгах

zerotier.com/download руу орж татаж ав.

  • Windows дээр .msi файлыг ажиллуулаад Next → Install.
  • Суусны дараа ZeroTier icon компьютерийн доод баруун булан (цагны хажууд) гарч ирнэ.
Icon харагдахгүй бол доод баруун булангийн ˄ дээр дарж далд icon-уудыг гарга. Mac дээр дээд баруун цэсэнд байна.
2

Сүлжээнд нэгдэх (хамгийн чухал)

  • Доод баруун булангийн ZeroTier icon дээр хулганы баруун товч дар.
  • Гарч ирсэн цэснээс “Join New Network…” дар.
  • Нээгдсэн нүдэнд доорх Network ID-г буулгаад “Join” дар:
233ccaac27e4c1dd
“Allow Managed / Allow Global” асуувал Yes / Allow. Сүлжээ public тул зөвшөөрөл хүлээхгүй — шууд холбогдоно.
3

CS2 дотор холбогдох

  • CS2 тоглоомоо нээ.
  • ~ товч дарж console нээ. Гарахгүй бол: Settings → Game → “Enable Developer Console” → Yes.
  • Доорх командыг бичээд Enter:
connect 172.24.238.93:27015
Серверт орно! 🎉 Нууц үг шаардахгүй.

  • ZeroTier ажиллаж байна уу? Icon дээр дарж сүлжээний хажууд ногоон / OK байгааг шалга.
  • Зөв Network ID? 233ccaac27e4c1dd — яг таарч байх ёстой.
  • Дээрх статус “ОФЛАЙН” бол сервер унтарсан — Ганбаад хэл, асаана.
