Амар холбогдоно
Нууц үг ч, зөвшөөрөл ч хэрэггүй — 3 алхмаар л ороод тоглочихно.

Яаж холбогдох вэ?

Ердөө 3 алхам. Нэг л удаа хийчихвэл дараа нь шууд ороод байх болно.

1

ZeroTier суулгах

Эндээс татаж ав → zerotier.com/download

  • Windows бол татсан .msi файлаа ажиллуулаад Next → Install дараад л суучихна.
  • Суусны дараа дэлгэцийн доод баруун буланд (цагныхаа хажууд) ZeroTier-ийн icon гарч ирнэ.
Хэрэв icon харагдахгүй бол доод баруун булан дахь ˄ сум дээр дар — нуугдсан icon-ууд гарч ирнэ. Mac дээр бол дэлгэцийн дээд баруун буланд байгаа.
2

Сүлжээндээ нэгдэх (хамгийн чухал нь энэ)

  • Доод баруун булан дахь ZeroTier icon дээрээ хулганы баруун товч дар.
  • Гарч ирэх цэснээс «Join New Network…» гэснийг дар.
  • Гарч ирэх нүдэнд доорх Network ID-г хуулж тавиад «Join» дар:
233ccaac27e4c1dd
«Allow Managed» гэх мэт зүйл асуувал бүгдэд нь Yes / Allow гэж хариулаарай. Манай сүлжээ нээлттэй болохоор хэн нэгний зөвшөөрөл хүлээхгүйгээр шууд холбогдчихно.
3

CS2 дотроосоо холбогдох

  • CS2-гоо нээ.
  • ~ (тильд) товчоо дараад console-оо гарга. Гарахгүй бол Settings → Game → «Enable Developer Console»-г Yes болго.
  • Дараа нь доорх командыг бичээд Enter дар:
connect 172.24.238.93:27015
Тэгээд л серверт орчихно! 🎉 Нууц үг ч хэрэггүй.

Тоглоом дотор — командууд

Командуудыг чат дотор бичнэ: Y дараад бичээд Enter.

⚠️ Холбогдож чадахгүй байна уу?
  • ZeroTier чинь асаалттай юу? Icon дээрээ дараад манай сүлжээний хажууд ногоон OK гэж байгаа эсэхийг хараарай.
  • Network ID зөв үү? 233ccaac27e4c1dd — үсэг тоо нэг бүрчлэн таарч байх ёстой шүү.
  • Дээр «ОФЛАЙН» гэж байвал сервер унтраастай байна — Ганбаад хэлээрэй, асаачихна.
  • Console гарч ирэхгүй бол CS2 → Settings → Game → «Enable Developer Console»-г Yes болго.
  • Windows Firewall ZeroTier-ийн талаар асуувал заавал Allow гэж хариулаарай.
