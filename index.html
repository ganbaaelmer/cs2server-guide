Хэрхэн холбогдох вэ?

Ердөө 3 алхам. Эхний удаад л тохируулна — дараа нь шууд ордог.

1

ZeroTier програм суулгах

zerotier.com/download руу орж татаж ав.

  • Windows дээр .msi файлыг ажиллуулаад Next → Install дар.
  • Суусны дараа ZeroTier icon компьютерийн доод баруун булан (цагны хажууд) гарч ирнэ.
Icon харагдахгүй бол доод баруун булан дахь ˄ (жижиг сум) дээр дарж далд icon-уудыг гаргана. Mac дээр бол дэлгэцийн дээд баруун цэсэнд байрлана.
2

Сүлжээнд нэгдэх (энэ хамгийн чухал)

  • Доод баруун булан дахь ZeroTier icon дээр хулганы баруун товч дар.
  • Гарч ирсэн цэснээс “Join New Network…” дээр дар.
  • Нээгдсэн цонхны нүдэнд доорх Network ID-г буулгаад “Join” дар:
233ccaac27e4c1dd
“Allow Managed Addresses / Allow Global” гэж асуувал Yes / Allow гэж хариулаарай. Сүлжээ public тул зөвшөөрөл хүлээх шаардлагагүй — шууд холбогдоно.
3

CS2 дотор холбогдох

  • CS2 тоглоомоо нээ.
  • ~ (tilde, Esc-ийн доор) товч дарж console нээ. Console гарахгүй бол: Settings → Game → “Enable Developer Console” → Yes.
  • Доорх командыг бичээд Enter дар:
connect 172.24.238.93:27015
Серверт орно! 🎉 Нууц үг шаардахгүй.

Тоглоом дотор — командууд

Бүгд чат дотор бичигдэнэ: Y дараад бичээд Enter.

⚠️ Холбогдож чадахгүй бол?
  • ZeroTier ажиллаж байна уу? Доод баруун булангийн icon дээр дарж сүлжээний нэрийн хажууд ногоон / OK байгаа эсэхийг шалга.
  • Зөв Network ID оруулсан уу? 233ccaac27e4c1dd — үсэг тоо яг таарч байх ёстой.
  • Дээрх статус “ОФЛАЙН” байвал — сервер унтарсан байна. Ганбаад хэл, асаана.
  • Console гарахгүй байна уу? CS2 → Settings → Game → “Enable Developer Console” = Yes.
  • Windows Firewall ZeroTier-г асуувал Allow дар.
